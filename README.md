# Electric Vehicle Price Analysis

## Project Overview

This project analyzes electric vehicle (EV) price data to understand pricing patterns across different brands and models. The goal is to explore the dataset, identify trends in EV pricing, and generate insights about the electric vehicle market.

## Objectives

* Analyze electric vehicle prices across different brands and models.
* Understand variations in EV prices across markets.
* Explore relationships between vehicle specifications and price.
* Generate insights about EV market trends.

## Dataset

The dataset contains information about electric vehicles including brand names, vehicle models, and price details.

Main fields in the dataset include:

* Brand
* Vehicle Model
* Price information

## Tools Used

Python
Pandas
NumPy
Matplotlib
BeautifulSoup
Requests

## Analysis Performed

* Data collection using web scraping
* Data cleaning and preprocessing
* Exploratory data analysis
* Price comparison between EV brands
* Visualization of EV price distribution

## Conclusion

The analysis helps in understanding EV pricing trends and brand differences in the electric vehicle market. These insights can help in studying market strategies and consumer price patterns.
